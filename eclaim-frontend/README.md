# Next.js




## Overview

This repository hosts the source code for the E-Claim application, which interacts with the blockchain to manage and verify claims. Users can submit claims, track their status, and retrieve claims using various filters for easy search and verification.

## Installation
1. Clone the repository:
```bash
 git clone https://github.com/saqibaltafapeiro/eclaim-testnet
```

2. Install dependencies:
```bash
 npm install
 ```

 2. Run the Server:
```bash
 npm run dev
 ```
